1. What is HTML? What does it do?
HTML is a markup, standard language used to create and structure content on the web. It provides a framework for organizing text, images, links, and multimedia on webpages.

2. Access https://www.fordham.edu/. In your own words, describe how this webpage uses structure to convey meaning and navigation options to the user.
A hierarchical structure is used on the Fordham University website to display information in an understandable manner. A navigation bar at the top provides links to important areas such as "Academics," "Admissions," and "Campus Life." The homepage makes it simple for users to find pertinent information by using headings, images, and sections to lead them through its content. Navigation is further improved by interactive features like call-to-action buttons and dropdown menus.

3. What is an HTML element? Tag? Attribute?
Tag: An element's defining markers. Tags are enclosed in angle brackets, like
for a paragraph.
HTML Element: The building block of HTML, comprising a start tag, content, and an end tag. For example, <p>This is a paragraph.</p> is a paragraph element.
Attribute: Additional information provided inside the start tag to modify an element’s behavior or appearance. For example, in <img src="image.jpg" alt="description">, src and alt are attributes.

4. What does it mean when an element has semantic meaning? And structural meaning?
When an element has semantic meaning, it will communicate the kind of content it contains. With structural meaning, it refers to how elements organize content on a page.

5. What is the default behavior when submitting a HTML form?
When an HTML form is submitted, the browser sends the form data to the server specified in the action attribute using the method defined in the method attribute (the default would be get).
